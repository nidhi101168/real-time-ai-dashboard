# PROJECT OVERVIEW

## 1. Project Name
Real-Time AI Dashboard (MERN + MySQL)

## 2. Project Objective
Build a production-style SaaS dashboard with:
- secure authentication
- role-based access control
- real-time communication
- AI-powered insights
- analytics
- caching
- containerized deployment
- CI/CD pipeline

This project is built step-by-step to learn real-world MERN architecture.

---

## 3. Tech Stack

### Frontend
- React
- React Hooks
- Context / Zustand (later)
- Tailwind CSS

### Backend
- Node.js
- Express.js
- JWT Authentication
- Role-Based Access Control (RBAC)

### Database
- MySQL

### Real-Time
- WebSockets
- Socket.io

### Caching
- Redis

### AI
- OpenAI API

### DevOps
- Docker
- Docker Compose
- GitHub Actions (CI/CD)

---

## 4. Core Features

1. User Authentication (JWT + Refresh Token)
2. Role-Based Access Control (Admin, Manager, User)
3. User Management
4. Real-Time Notifications & Chat
5. Analytics Dashboard (MySQL Aggregations)
6. AI Insights using OpenAI
7. Redis Caching for Performance
8. Dockerized Deployment
9. CI/CD Pipeline
